### 01. 단위 변환기 CLI

Rust Book 초반 챕터(변수, 함수, 제어 흐름)를 학습한 직후 1~2시간 안에 만들어 보는 가장 작은 단위의 CLI 프로젝트입니다. 표준 라이브러리만 사용하며, 외부 크레이트를 추가하지 않는 것을 목표로 합니다.

#### 선행 챕터
rustlings 기준으로 다음 세 챕터를 마친 직후가 적합합니다.

- `01_variables` — `let` / `let mut`, 섀도잉, 타입 어노테이션
- `02_functions` — 함수 시그니처, 매개변수, 반환값(`->`), 표현식 vs 문장
- `03_if` — `if` / `else`, 표현식으로서의 `if`

#### 기능 명세
명령줄 인자로 값과 단위를 받아 다른 단위로 변환한 결과를 출력합니다.

```bash
cargo run -- <값> <단위>
```

지원하는 변환은 다음 두 쌍입니다.

| 입력 단위 | 출력 단위 | 비고                           |
|-----------|-----------|--------------------------------|
| `c`       | °F        | 섭씨 → 화씨                    |
| `f`       | °C        | 화씨 → 섭씨                    |
| `kg`      | lb        | 킬로그램 → 파운드              |
| `lb`      | kg        | 파운드 → 킬로그램              |

변환 공식

- 섭씨 → 화씨: `F = C × 9 / 5 + 32`
- 화씨 → 섭씨: `C = (F − 32) × 5 / 9`
- kg → lb:   `lb = kg × 2.2046226218`
- lb → kg:   `kg = lb / 2.2046226218`

#### 핵심 활용 포인트
이 프로젝트에서 의식적으로 연습할 Rust 요소들입니다.

- `let` / `let mut` — 인자에서 읽어 온 값의 가변/불변 구분
- 함수 시그니처와 반환값 — 변환 함수마다 `fn name(x: f64) -> f64` 형태로 분리
- `if` / `else` — 단위 문자열에 따른 분기
- `match` 맛보기 — `if/else` 체인 대신 단위 문자열을 매칭하는 형태로 리팩터링해 보기
- `f64` 산술 연산과 `println!`의 포맷 지정자(`{}`, `{:.2}` 등)

#### 검증 예시
다음 두 케이스가 기대대로 동작하면 1차 목표 달성으로 간주합니다.

```bash
cargo run -- 0 c     # → 32 °F
cargo run -- 100 lb  # → 약 45.36 kg
```

#### 디렉토리 구조

```
my_01_unit_converter/
├── Cargo.toml
├── README.md
└── src/
    └── main.rs
```

#### 실행 방법

```bash
# 빌드만
cargo build

# 한 번에 빌드 + 실행 (인자는 -- 뒤에)
cargo run -- 100 c

# 릴리스 빌드
cargo build --release
./target/release/my_01_unit_converter 100 c
```
